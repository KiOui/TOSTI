TOSTI MCP server

Model Context Protocol endpoint for AI assistants

This URL is the Model Context Protocol endpoint for TOSTI. It is meant to be consumed by an AI assistant, not opened in a browser.

If you're a regular user trying to hook up your assistant, see the Connect an AI assistant explainer.

For developers

/mcp implements the Model Context Protocol over streamable HTTP, gated by OAuth 2.0. Send POST requests with Accept: application/json, text/event-stream; this landing page is only served for browser-style GETs.

Auth details (supported grants, scopes, dynamic client registration, PKCE, redirect URIs) are in the OAuth integration guide. The discovery documents at /.well-known/oauth-authorization-server and /.well-known/oauth-protected-resource are the source of truth.

The full list of tools exposed by this server — with input/output JSON Schemas — is at /mcp/docs/. To explore them interactively, run MCP Inspector locally with npx @modelcontextprotocol/inspector and point it at this URL.
