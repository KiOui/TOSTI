How to process deposit (statiegeld)?

Help recycling cans!

Tartarus sells items that sometimes have a deposit on them (for example, the tin cans with soda). These cans can be handed in at the counter for recycling. If a user hands in a used can with deposit, they receive a balance in T.O.S.T.I. that can be used for discount on future purchases. This guide will show you how to process a user that wants to hand in a deposit.


1. Go to a running shift

Deposit can only be added or used if a shift is open. If you are not sure how to enter a shift, please refer to the "How to manage a shift?" tab above. Make sure you are in the administrator view of a running shift.


2. Let the user open an account

Before the user can hand in a deposit, they need to open an account in T.O.S.T.I. Users can open an account by going to the "Transactions" tab under the Account view.


3. Let the user open their personal QR code

The user should now open their personal QR code by pressing the button in the header.


4. Open the QR code scanner

Now you can open the scanner on the shift administrator screen by pressing the Scanner button on the admin footer.

A screen will popup asking for permission to use the camera of your device. Press accept to start the scanner. You can now scan the QR code of the user with your camera.


5. Add a transaction

A new screen will pop up, showing the name of the user and the current balance of the user. You are now able to add or subtract balance by adding a new transaction to the user's account. Once you add a transaction, the popup will update and the new balance of the user will be shown.

When a user hands in some deposit you can add this to their account, when a user wants to use their deposit you can remove some of their deposit from their account.

After adding the transaction you can manually give the user a discount on their payment.
