How to manage and create shifts?

Before you can start managing shifts on TOSTI, first login using your Science account via the login page. Then, go to the home page to see if there is an active shift available which you can join. The buttons on the home page can state two things.

  • Join shift at ...: There is already an active shift created at the venue, you can choose to join this shift as a shift administrator.
  • Start shift at ...: There is no active shift for the venue at this moment, you can choose to create a shift.

Joining a shift

Joining a shift is easy, just press the button and confirm you want to join the shift.

Starting a shift

In order to start a new shift, press the button on the home page to start a shift for a specific venue. A form will pop up requiring you to enter some details regarding the shift you are about to start. Most values are already filled in. You might want to change the assignees of the shift to the people that need to have administrator rights for the shift.
