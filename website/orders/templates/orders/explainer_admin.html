How to manage and create shifts?

Everything you need to know as a baker.


1. Log in

Before you can start managing shifts, first login using your RU account.


2. Start or join a shift

After you are logged in, you can either start a new shift or join a shift in a venue you are authorized for, via the buttons on the the home page.

When starting a new shift, you can set the begin and end time of the shift. After creating the shift, it will appear on the site. Note that there cannot be 2 shifts for the same venue at the same time.

If you want to join an already existing shift, you can simply click 'join shift'.

When you have joined a shift as a shift admin, you will see the shift admin page. This is where all orders will come in and where you can register which orders are paid or ready. Via the footer menu, you have access to some quick actions, like increasing the number of orders you accept.


3. Allow orders

A new shift won't directly accept new orders. You need to allow (or disallow) orders via the -button in the footer menu. Only when orders are allowed, people can place orders. Otherwise, a message will be shown that ordering products is currently not possible for this shift.

Note that even when orders are allowed, the maximum order-capacity for the shift or the end-time constraints are in place.


4. Wait...

Wait for orders to come in. New orders will automatically appear in the queue.

Of course, you can place orders yourself as well. Those will automatically move to the top of the queue. Via the -button in the footer menu, you can easily go to the order page to place an order yourself.


5. Process orders

When orders start coming in, you can start preparing them.

If an order is paid, you can hit the Paid button to turn it into Paid.

When an order is ready, hit the Ready button to mark it as Ready. You can now call out the person's name who placed the order to pick it up.

Via the -button in the footer menu, you can get an overview of all products you still have to make.


6. Scan products

There are also products that do not have to be prepared but rather can be scanned.

You can start scanning a product with a barcode by pressing the Scanner button on the admin footer.

A screen will popup asking for permission to use the camera of your device. Press accept to start the barcode scanner. You can now scan a product with a barcode, if the barcode is scanned it will be automatically added to the queue. Scanned orders will appear with the user marked as .

You can also search for product to add manually via the search bar above the scanner.

If you want to remove a scanned order you can press the Remove button on the right of a scanned order. Accept the popup and the order will be deleted from the queue.


7. End of your shift

When the end-time of your shift has passed, orders will automatically stop coming in. Also, when the maximum capacity is reached, orders will stop coming in. If you want to increase the maximum capacity or postpone the end-time, you can do so via the + 5 or + 5 buttons.


8. Closing your shift

After all the orders are processed and the shift has come to an end, you must finalize the shift. Finalizing makes a shift uneditable and closes it permanently. Make sure that all orders are Paid and Ready first. Click the Finalize button in the footer to finalize a shift. Thanks for baking!

If you want to change more advanced details, use the -button to go to the detailed admin view. This button is only visible if you are authorized for this.

Have fun baking!
